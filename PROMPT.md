# Agent Prompt

Copy this prompt into your coding agent.

```text
请基于这个 GitHub 项目，帮我在本地复刻一个火烧云 / 朝霞 / 晚霞预测网页。

目标：
1. 保留单文件网页结构，优先让它能在本地直接运行。
2. 保留地图、城市选择、未来 7 天、朝霞/晚霞切换、火烧云指数、天气预报、手机端适配。
3. 使用 Open-Meteo 免费天气 API 获取云量、分层云量、降水、能见度、湿度、风速、日出日落时间。
4. 使用 MapLibre GL 展示地图。中文用户默认中文地图；海外用户可以用 ?map=en 切换英文 OpenStreetMap。
5. 算法要区分两件事：
   - 本地点天气：看降水、低云、能见度、风速。
   - 太阳方向云层：晚霞看西侧，朝霞看东侧，对 35km、90km、160km 三段云层做加权评估。
6. 如果 API 暂时失败，要保留离线估算兜底，避免页面空白。
7. 手机端打开后要有科技感的磨砂玻璃信息卡，地图点要清晰可点。

请先阅读 index.html 和 docs/algorithm.md，再修改。不要引入复杂框架，保持项目适合初学者复制。
```
